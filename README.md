# Proyecto Usuarios CRUD

Aplicación web simple que permite crear, listar, editar y eliminar usuarios.

## Tecnologías

- HTML
- CSS
- JavaScript (Vanilla)
- API REST (Java backend)
- Base de datos (PostgreSQL)

## Funcionalidades

- Listar usuarios
- Crear usuario
- Editar usuario
- Eliminar usuario
